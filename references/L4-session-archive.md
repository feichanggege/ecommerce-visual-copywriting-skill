### 项目3：欧诗芙·紧致抗皱面膜

- **日期**：2026-04-15
- **产品类型**：化妆品·妆字号（待确认是否有特证/功效评价报告）
- **品牌**：欧诗芙 OSFO
- **产品全称**：欧诗芙紧致抗皱面膜
- **企业**：广昀星细胞技术有限公司
- **专利号**：ZL2022221508890.3（专利成分：透明质酸钠）
- **核心成分群**：
  - 四重透明质酸钠形态（普通型/水解型/乙酰化型/交联聚合物型）
  - 三大植物愈伤组织培养物（海茴香+高山火绒草+苹果细胞）
  - 白细胞提取物 + 丝心蛋白 + 烟酰胺 + 泛醇
  - 不含酒精（乙醇）
- **目标人群**：35-65岁女性，关注护肤/睡眠/内调
- **文案策略**：场景情绪型（妆字号不碰功效，走场景+体验+感受）
- **必卖理由（确认版v6）**：
  1. 忙了一天脸很累 → 敷一片给自己放松 ★★★★★
  2. 35岁以后脸不如从前 → 日常好好照顾自己 ★★★★
  3. 晚上敷着敷着就睡着了 → 客户反馈有助睡眠 ★★★★
- **交付物**：`欧诗芙面膜_主图详情页执行方案.md`（主图5张+详情M1-M4）
- **关键决策**：
  - ❌ 最初方向：讲专利号+化学成分 → 用户否决："买家看不懂有什么用"
  - ✅ 转向：用买家语言写必卖理由（问题→解决方案），成分退到信任证据位
  - ✅ 再次转向：加入客户反馈（"敷着就睡着了"），借客户嘴说体验
  - 核心方法论：**不碰功效 ≠ 没东西写 → 场景+情绪+体验+生活方式**
- **踩坑记录**：
  - 专利号一开始写错（ZL2014... 实际是 ZL2022...）→ 用户纠正
  - 博士团队被移除出必卖理由 → 不是买家的购买驱动
  - 初版全是成分和技术术语 → 完全偏离买家视角
  - 用户反复强调："要用买家理由来输出"
- **可复用经验**：
  - **妆字号化妆品黄金公式**：场景代入 + 情绪共鸣 + 使用体验 + 客户证言引用
  - **买家语言vs卖家语言**：买家不在乎专利号化学名，只关心"跟我有什么关系"
  - **客户反馈是合规神器**：借客户嘴说感受 ≠ 产品自宣功效承诺
  - **"敷着就睡着了"模式**：不说"助眠"，说客户实际使用中的自然结果
- **待改进**：
  - 品牌主色值未确认（设计说明中缺）
  - 尚未拿到功效评价报告 → 如果后续拿到可升级加入功效线
  - 详情页只做了M1-M4，M5-M8是否需要待定
  - 太岁水和贪睡宝（同品牌其他产品）尚未做
