## 四、必卖理由模式库

### 模式A：痛点 → 产品属性解决
```
适用：有明确技术差异的产品
结构：人群痛点 → 技术事实（专利/成分/工艺） → 利益翻译
例：杞明枸杞籽油 → 免泡煮即食（工艺事实） → 忙碌也能吃上
```

### 模式B：痛点 → 场景+情绪解决
```
适用：妆字号/普通食品（不能说功效）
结构：生活痛点 → 使用场景 → 情绪价值 → 客户证言
例：欧诗芙面膜 → 下班脸累 → 敷15分钟放松 → 爱自己 → 敷着睡着
```

### 模式C：痛点 → 安全安心解决
```
适用：用户顾虑安全性/不敢尝试的产品
结构：怕X → 成分透明/资质备案/不含XX → 用着放心
例：欧诗芙第②条 → 怕烂脸 → 成分清楚无酒精 → 安心
```

### 模式D：痛点 → 习惯养成解决
```
适用：需要长期坚持的产品
结构：想改变但总拖延 → 每天一件小事 → 时间给答案
例：欧诗芙第②条 → 总等下次 → 每天一片 → 坚持就有变化
```
