## 零层：通用规则（所有产品必过）

> **法律依据**：以下所有通用规则均源自《广告法》第二章广告内容准则（第8-17条）及第五章法律责任，适用于**所有品类**，无例外。

### 1. 绝对化用语禁用清单（§9第3项）
**禁止**：唯一、最、所有、只有、第一、绝对、顶级、极致、完美、无敌、彻底、完全、100%、99%、100倍、**国家级、最高级、最佳**（§9第3项原文禁止词）、特效
**替换为**：多数 / 约 / 显著 / 较为 / 之一 / 部分 / 大多数 / 助力 / 逐步

#### 1.1 分类禁用词明细（2025年违禁词汇总补充）

**「最」类**：最佳、最具、最爱、最赚、最优、最优秀、最好、最大、最大程度、最高、最高级、最高档、最奢侈、最低、最低级、最低价、最便宜、最流行、最受欢迎、最时尚、最舒适、最先、最先进、最新、最新科技、最新科学、最新技术、最后

**「级/极」类**：国家级产品、全球级、宇宙级、世界级、顶级、顶尖、尖端、高级、极品、极佳、绝佳、绝对、终极、极致

**「首/家/国」类**：首个、首选、独家、独家配方、首发、全网首发、全国首发、首次、首款、全国销量冠军、填补国内空白、中国驰名（驰名商标）、国际品质

**「品牌」类**：大牌、金牌、名牌、王牌、领袖品牌、世界领先、遥遥领先、领导者、缔造者、创领品牌、巨星、著名、掌门人、至尊、巅峰、奢侈、资深、领袖、之王、王者、冠军

**「权威」类**：老字号、中国驰名商标、特供、专供、专家推荐、质量免检、无需国家质量检测、免抽检、国家XX领导人推荐、国家XX机关推荐

**「虚假」类**：史无前例、前无古人、永久、万能、祖传、纯天然、100%、高档（无依据时）、超赚、精准

**迷信用语**（⚠️ 新增）：带来好运气、增强第六感、化解小人、增加事业运、招财进宝、健康富贵、提升运气、护身、平衡正负能量、消除精神压力、调和气压、逢凶化吉、时来运转、万事亨通、旺人、旺财、助吉避凶、转富招福

**诱导消费话术**（⚠️ 新增）：秒杀、抢爆、再不抢就没了、不会更便宜了、错过就没机会了、万人疯抢、全民疯抢/抢购、卖疯了、抢疯了

**限时促销禁用表述**（⚠️ 新增）：随时结束、仅此一次、随时涨价、马上降价、最后一波（须有具体时限方可使用促销话术）

### 2. 国旗/国徽/国歌/军旗/国家机关名义（§9第1、2项）⚠️ 新增
- **绝对禁止**使用或变相使用：国旗、国歌、国徽、军旗、军歌、军徽
- **绝对禁止**使用国家机关或国家机关工作人员的名义/形象为产品站台
- 常见违规表现：「国家XX部推荐」「XX局认证」「官方指定」「政府采购」「央X推荐」
- ⚠️ 注意：「官方」二字本身极易误导，如无真实政府授权资质，**谨慎使用或回避**

### 3. 广告可识别性（§14）⚠️ 新增
- 广告必须具有可识别性，使消费者能辨明其为广告
- 电商平台规则要求：信息流广告/种草笔记/直播带货须标明「广告」或「推广」字样
- 通过「知识科普」「健康养生」「使用教程」等内容形式**变相发布产品广告，构成违法**（§14第2款、§17）
- 特别注意：以介绍健康、养生知识等形式**变相发布医疗、药品、保健食品广告**，构成虚假广告（§17）

### 4. 数据/事实宣称必须有背书（§11）
- 每个"XX倍""XX%""未检出""含有" → **必须附第三方检测报告编号**
- 使用调查结果、统计数据、文摘、引用语 → **必须注明真实出处**（§11第2款）
- 引证内容有**适用范围和有效期限**的 → **必须明确表示**（§11第2款）
- 认证类宣称 → **必须附认证机构全称+证书编号**
- 无报告支撑的数据 → 删除或改为定性模糊描述（"含量较高""多数情况下"）

### 5. 专利宣称规范（§12）
- 涉及专利产品或专利方法 → **必须标明专利号和专利种类**
- 未取得专利权的 → **不得谎称已取得专利权**
- 禁止使用：专利申请中（未授权）/ 已终止、撤销、无效的专利
- 常见违规：「专利技术/专利成分」但无证书编号 → 必须提供真实有效专利号

### 6. 竞品对比原则（§13）
- **不直接贬低竞品品牌**（§13明文禁止）
- **不做"我优他劣"的二元对立表述**
- 正确方式：陈述自身属性 + 引用行业公开数据做客观参照

### 7. 广告代言人规范（§38）⚠️ 新增
- 广告代言人**不得为其未使用过的商品/未接受过的服务**作推荐、证明
- **医疗、药品、医疗器械广告中不得使用广告代言人**（绝对禁止）
- **保健食品广告中不得使用广告代言人**（§18第1款第5项）
- 代言人明知广告虚假仍代言 → 没收违法所得并处罚款（§61）

### 8. 虚假广告的法律定义（§28）⚠️ 新增
以下五种情形认定为**虚假广告**，最高可处广告费用5倍以上10倍以下罚款+吊销营业执照：
1. 商品/服务不存在的
2. 商品性能、功能、产地、用途、质量、成分、价格、生产者等信息与实际情况不符，对购买行为有实质影响的
3. 使用虚构、伪造或者无法验证的科研成果、统计资料、调查结果、文摘、引用语等信息作证明材料的
4. 虚构使用商品或者接受服务的效果的
5. 以虚假或者引人误解的内容欺骗、误导消费者的其他情形

### 9. 价格合规红线（⚠️ 2026新增，覆盖所有品类）

> **法律依据**：《价格法》+《明码标价和禁止价格欺诈规定》（总局令第56号，2022.7.1施行）+《互联网平台价格行为规则》（三部门联合发布，2026.4.10实施）

#### 9.1 划线价/被比较价格规则（总局令第56号 §16-17）
- 划线价必须**真实准确**，有依据可查
- 未标明被比较价格时，划线价**不得超过前7日内最低成交价**；前7日无交易则不得超过**前最后一次交易价格**
- **禁止使用无依据或无从比较的价格**作为折扣计算基准
- 与厂商建议零售价比较时，**必须明确标注「厂商建议零售价」**，变动时须立即更新
- **禁止先涨后降**：无合理理由，不得在折价/减价前临时显著提高标示价格

#### 9.2 价格欺诈认定（总局令第56号 §19）
以下行为构成**价格欺诈**，可处5万元以下罚款；情节严重的5万-50万元：
- 谎称价格为政府定价或政府指导价
- **低价诱骗消费者，高价结算**（首页价低于详情页、结算价高于展示价）
- 通过**虚假折价、减价或价格比较**销售
- 使用欺骗性、误导性的语言、文字、数字、图片标示价格
- 无正当理由**拒绝履行或不完全履行价格承诺**
- **不标示或显著弱化标示**对消费者不利的价格条件
- **大数据杀熟**：基于用户画像实施差别定价（2026年《互联网平台价格行为规则》明确禁止）

#### 9.3 促销价格合规（所有品类必做）
- 促销须标明**活动规则、期限、适用范围**
- 赠品须标明**品名、数量**；标有价格/价值的须为**当前实际销售价格**
- **限时促销必须有明确时限**：❌「随时结束」「最后一波」「仅此一次」→ ✅「XX月XX日-XX月XX日」
- 积分/礼券抵扣规则须**完整明示**

#### 9.4 电商平台专属价格红线
- **首页/搜索页价格不得低于详情页价格**（强制）
- 促销活动范围、规则须与实际一致
- 平台不得以流量限制/搜索降权胁迫商家参与低价促销
- 禁止要求商家低于成本销售

#### 9.5 违规替代词

| ❌ 禁用价格话术 | ✅ 合规替代 |
|---------------|-----------|
| 最低价、最便宜、全网最低 | 限时优惠、超值特惠 |
| 原价XX、原价 | 建议零售价XX（须为真实建议零售价） |
| 随时涨价、仅此一次 | XX月XX日前有效、限时活动 |
| 抢疯了、万人疯抢 | 热销中、用户好评 |
| 跳楼价、吐血价 | 限时折扣、限时让利 |

### 10. 反不正当竞争法·电商专项（⚠️ 2025修订版新增）

> **法律依据**：《反不正当竞争法》（2025年修订版，2025年施行）

#### 10.1 虚假/引人误解的商业宣传（§8）
- 不得对商品的性能、功能、质量、销售情况、用户评价、曾获荣誉、资格资质等作**虚假或引人误解的商业宣传**
- 违者处**20万元以上100万元以下罚款**；情节严重的100万-200万元，可吊销营业执照

#### 10.2 帮助虚假宣传（§9第2款，⚠️ 2025新增）
- **不得通过组织虚假交易、虚假评价等方式，帮助其他经营者进行虚假或引人误解的商业宣传**
- 2025修订版明确将**刷单组织者/水军公司/刷单平台**纳入违法主体
- 违者依§25处罚（10万元起步）

#### 10.3 电商常见违规情形
| 违规类型 | 示例 | 法律依据 |
|---------|------|---------|
| 虚构交易数据 | 「月销10万+」「全网爆款」无真实销量支撑 | §8 |
| 虚假用户评价 | 好评返现后操控评价；组织刷好评 | §8+§9 |
| 虚构荣誉资质 | 伪造获奖证书/行业协会推荐 | §8 |
| 夸大销售状况 | 「500万人都在用」「销量遥遥领先」 | §8 |
| 编造产地信息 | 进口产品谎称原装进口；农产品谎称特定产区 | §8 |

### 11. 直播电商合规（⚠️ 2026新增）

> **法律依据**：《直播电商监督管理办法》（总局令第117号，2026.2.1施行）

#### 11.1 直播带货基础合规
- 直播页面须**显著展示**商品品名、价格、计价单位
- 促销方式须标明**计算基准**（划线价须为真实有效价格）
- 每次直播前须**事前合规审核**展示内容、讲解内容、服装、布景、道具
- 核验商品经营者资质、产品合格证明等，记录保存**不少于3年**

#### 11.2 禁止行为
- 不得对商品性能、功能、质量、销售情况、用户评价等作**虚假或引人误解的商业宣传**
- 不得销售掺杂掺假、以假充真、以次充好的质量不合格商品
- 不得发布虚假广告

#### 11.3 AI直播专项（§37，⚠️ 新业态）
- 使用AI生成人物图像/视频从事直播，须符合法律法规和强制性标准
- 须依照国家有关规定进行**标识**，持续向消费者提示该内容由AI生成
- 违规责任由管理或使用该AI内容的**直播间运营者**承担
- 平台须防范利用AI技术编造虚假商业信息、假冒他人进行商业宣传

#### 11.4 虚假宣传处罚
- 虚假宣传 → 依照《反不正当竞争法》处罚
- 发布虚假广告 → 依照《广告法》处罚（最高广告费5-10倍罚款）
- 帮助虚假宣传（刷单/刷评）→ 10万元起步罚款

### 12. 互联网广告可识别性（2024执法指南细化版）

> **法律依据**：《互联网广告可识别性执法指南》（2024年8月发布）+《互联网广告管理办法》（2023年3月施行）

#### 12.1 核心判定标准（⚠️ 关键更新）
- **是否标明「广告」不是判断它是不是广告的依据**——只要实质上构成推销商品或服务，就是广告
- 「图文种草」「视频测评」「知识科普」**+ 挂车/放购物链接 = 必须显著标明「广告」**
- 达人/博主发布此类内容 → **直接认定为「广告发布者」**，需承担法律责任
- 不能以「个人经验分享」为由规避广告法约束

#### 12.2 可识别性的实现方式
- **文字标注**：必须**显著标明「广告」**（字号、颜色须醒目）
- **语音标注**：必须**清晰语音提示「广告」**
- **设立专门广告区**：如区域内显著标明「广告」或明确告知均为广告
- **直播带货**：直播间始终显著标明其为商品经营者，或直播页面显著标明为广告

#### 12.3 严禁变相发布
- 严禁以**新闻报道形式**变相发布广告
- 以介绍健康、养生知识等形式变相发布医疗/药品/保健食品广告 → **违法**

#### 12.4 平台连带责任
- 平台利用**人工干预、算法推荐**影响展示效果并构成广告的 → **平台认定为广告发布者**
- 竞价排名商品/服务 → **必须显著标明「广告」**

## 妆层：化妆品/面膜类（妆字号 & 特证）

> **核心前提**：市面上绝大多数面膜属于普通化妆品（妆字号），仅祛斑美白/防晒类属特殊化妆品，需取得《特殊化妆品注册证》方可宣传对应功效。

### 一、全品类通用·绝对禁止的红线（无论普通/特证，100%不能说）

#### 1. 广告法明令禁止的绝对化用语
**核心违禁词**：最、第一、唯一、顶级、极致、100%、完美、国家级、世界级、最高级、全网第一、行业标杆、天花板、零风险、无效退款、永不反弹、一次见效、永久有效
**补充严控**：
- 「全网销量第一」— 无权威第三方数据支撑绝对不能说
- 「零添加」「纯天然」「无防腐**全面禁止** — 国家药监局明确化妆品不可能实现"零添加"，此类表述必然构成虚假宣传

#### 2. 医疗化/药品相关 · 最高危红线
《化妆品监督管理条例》明确：**化妆品禁止宣称医疗作用，禁止使用医疗术语，禁止与医药/医疗机构关联。**

| ❌ 禁止类型 | 禁用词举例 |
|-----------|-----------|
| 治疗动词 | 治疗、修复、消炎、抗炎、杀菌 |
| 药妆关联 | 药用、药妆、医学护肤品、医美面膜、医用面膜、医用冷敷贴 |
| 医疗机构 | 医院同款、医生推荐、三甲医院认证、皮肤科专用、医美后专用 |
| 病理表述 | 激素脸、皮炎、湿疹、红血丝**修复**、痘印**修复**、痤疮改善 |
| ⚠️ 注意 | "改善痘痘"也属于医疗化暗示，**全面禁止** |
| 治疗效果词（⚠️ 新增） | 治愈率、有效率、痊愈、治愈、根除、根治 |
| 无痛/无创词（⚠️ 新增） | 无痕、无创、不留疤、不留痕、无痛 |
| 副作用词（⚠️ 新增） | 无副作用、无依赖性、不过敏 |
| 效果保证词（⚠️ 新增） | 效果显著、效果明显、完美贴合、完美还原、长效、特效 |
| 次数承诺（⚠️ 新增） | 一次XX、快速XX（如一次美白、快速祛痘） |

#### 3. 无依据虚假夸大表述
- 无对应检测报告 → 不能说「孕妇可用/敏感肌专用/儿童可用」
- 无真实专利证书 → 不能说「专利成分/专利技术」

- 效果承诺 → 7天美白、24小时持久保湿、一次淡化细纹等

#### 4. 不正当竞争类
- 禁止贬低竞品：「秒杀同类、比XX品牌效果好、吊打同行」
- 禁止虚假素材：PS前后对比图、无真实依据的用户案例/达人推荐

### 一、蓝帽子批文管理

| 纯感官形态描述 | 膏体细腻、延展性好、无刺激气味、不含香精 |
| 日常使用场景（只描述场景，不关联功效） | 办公长时间用眼后涂抹眼周、熬夜后眼周护理、户外暴晒后眼周舒缓、日常眼周皮肤清洁护理 |
| 合规资质信息 | 如实标注备案文号、生产企业信息、执行标准、保质期、使用方法，**不得夸大资质效力** |
| 成分客观标注 | 如实标注所含成分，不宣称成分功效，仅可说明「添加XX植物成分」 |

> **场景描述铁律**：只描述"什么时候用"，不描述"用了会怎样"。例：
> - ✅ 「办公后眼周护理使用」
> - ❌ 「长时间用眼后眼周舒缓（暗示缓解疲劳）」

### 三、资质标注规范

- **备案文号**：如实标注「豫促健认字（2XXX）第XXXX号」或对应省级协会备案号，**不得省略**
- **备案主体说明**：可附注「本产品备案主体为XX省XX行业协会，非国家行政部门许可」（主动说明有助于建立信任，规避误导嫌疑）

- **生产许可证**：如实标注，与包装信息完全一致

### 六、极高风险预警（上线前必读）

1. **资质认可风险**：「豫促健认字/用字」仅为行业协会自律备案，非政府部门行政许可，**部分平台会将其判定为「无正规资质产品」**，存在上架后被无故下架、清退的风险，建议上架前与平台小二确认资质认可情况。

2. **监管强化风险**：眼部产品为**市场监管总局年度重点整治品类**，哪怕是轻微擦边宣传，也极易触发投诉、举报、立案调查，后果严重。

3. **刑事风险线**：若产品实际为入眼使用，无论宣传是否合规，均属于违法产品，面临：
   - 没收违法所得
   - 最高货值金额20倍罚款
   - 情节严重的追究刑事责任
   
   **建议：上线前务必确认产品使用方式仅为外用，并在包装和文案中明确标注「仅供外用，不可入眼」。**

## 二层：运动器材/体态管理类（非医疗器械）

### 核心风险
极易触碰「医疗化暗示」「治疗效果承诺」红线——因为目标用户本身就是有身体困扰的人群

### 医疗化术语替换表（高风险，必须逐条检查）

| ❌ 禁止类型 | ❌ 禁用词举例 | ✅ 替换方向 | 原因 |
|-----------|-------------|-----------|------|
| 痛感描述 | 腰酸背痛、脖子痛、关节痛 | 腰背不适、肩颈紧绷感 | "痛"属病理症状 |
| 医学术语 | 骨盆前倾、脊柱侧弯、椎间盘 | 骨盆形态不佳、背部线条不直 | 医学术语暗示诊断 |
| 治疗动词 | 修复、治疗、康复、矫正 | 体态管理、体态调整、锻炼、训练 | 暗示能治疾病 |
| 效果承诺 | 恢复XX状态、找回XX身体 | 向XX状态靠近、改善当前状况 | 绝对化效果承诺 |
| 专业宣称 | 专业级、医疗级（无资质时） | 科学、系统、规范、严谨 | 无资质不可宣称 |

### 效果弱化原则（强制）
- 所有效果相关表述 **必须添加弱化词**：助力 / 逐步 / 辅助 / 有助于 / 因人而异
- ❌ "100%有效""彻底改善""一定见效"
- ✅ "助力逐步改善""效果因人而异"

### 免责声明模板
```
本产品为运动器材/健身设备，非医疗产品。
无法治疗疾病或病理问题。
体态改善/训练效果因人而异，
建议在专业人员指导下使用。
如有身体不适请及时就医。
```

## 三层：普通食品（非保健食品、非药品）⚠️ 最严

### 核心原则
**保健品至少有批文可说功能，普通食品什么功能都不能说。**

#### 四条底线（必须同时满足）
1. 无违禁绝对化用语
2. 无虚假宣传风险（所有卖点可验证）
3. 无不正当竞争风险（不贬低竞品）
4. 无医疗化暗示（全程未关联任何疾病/症状/功效）

#### 食品广告发布暂行规定·专用禁用词（⚠️ 新增）
> **法律依据**：《食品广告发布暂行规定》+《食品安全法》

| 禁用类型 | 具体禁用词/行为 | 说明 |
|---------|---------------|------|
| 绝对化用语 | 「最新科学」「最新技术」「最先进加工工艺」 | 食品广告专用禁用 |
| 虚假性词语 | 祖传、秘制 | 无依据的传承/配方宣称 |
| 夸大性词语 | 强力、特效、全效、强效、奇效、高效、速效、神效 | 暗示食品有治疗效果 |
| 混淆药品用语 | 「对XX疾病有预防和治疗作用」「降低血压」「防癌抗癌」 | 绝对禁止 |
| 医疗相关词语 | 处方、复方、治疗、消炎、活血、祛瘀、止咳、解毒、疗效、防治、增高、益智 | 全部禁止 |
| 替代母乳 | 暗示或明示可以替代母乳 | 不得使用哺乳妇女和婴儿形象 |
| 国家免检 | 「国家免检产品」字样和图案 | 该标识已废止，严禁使用 |
| 封建迷信 | 神丹、神仙、仙方等 | 绝对禁止 |

#### 食品标签合规（⚠️ 2025新国标）
> **法律依据**：《预包装食品标签通则》GB 7718-2025 +《预包装食品营养标签通则》GB 28050-2025（2025年发布，逐步实施）

- 电商详情页展示的营养成分数据**须与产品包装标签一致**
- 涉及营养成分的宣称（如「高钙」「低脂」「富含XX」）须符合GB 28050的营养成分含量声称标准
- 「0糖」「无糖」等声称须符合国标定义（≤0.5g/100g），且有检测报告支撑
- 电商页面不得展示与实际包装标签不一致的信息

#### 绝对不能做的事

| ❌ 禁止类型 | 示例 | 原因 |
|-----------|------|------|
| 功效暗示 | "喝走胀闷感""喝出好状态""日常调理" | 暗示身体变化=功效宣称 |
| 症状关联 | 描述胃胀/便秘/失眠/疲劳后接产品 | 痛点+产品=间接功效联想 |
| 人群疾病绑定 | "适合糖尿病患者""三高人群" | 将产品与疾病人群绑定 |
| 身体变化承诺 | "喝了之后XX""坚持饮用能XX" | 效果承诺无依据 |
| 原料功效转嫁 | "松花粉富含XX营养→所以能XX" | 用原料成分暗示成品功效 |

#### 安全表达白名单（只能说这些）

| ✅ 可以说 | ✅ 可以说 | ✅ 可以说 |
|-----------|-----------|-----------|
| 原料来源产地 | 工艺特点（水溶/速溶/无渣）| 口感风味描述 |
| 成分配料表 | 营养成分数据（需检测报告）| 饮用场景/时机 |
| 包装规格/便携性 | 品牌故事/源头把控 | 生产资质/卫生许可 |

#### 兜底声明（必须保留）
```
本品为普通食品，非保健食品，非药品。
不具有任何保健功能或治疗作用。
仅供日常食用。
```
