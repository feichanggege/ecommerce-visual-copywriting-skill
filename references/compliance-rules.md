>
> 适用法规：《广告法》《化妆品监督管理条例》《化妆品功效宣称评价规范》+淘宝/拼多多/抖音/京东全平台电商规则

## 零层：通用规则（所有产品必过）

### 绝对化用语禁用清单
**禁止**：唯一、最、所有、只有、第一、绝对、顶级、极致、完美、无敌、彻底、完全、100%、99%、100倍、国家级、特效
**替换为**：多数 / 约 / 显著 / 较为 / 之一 / 部分 / 大多数 / 助力 / 逐步

### 数据/事实宣称必须有背书
- 每个"XX倍""XX%""未检出""含有" → **必须附第三方检测报告编号**
- 认证类宣称 → **必须附认证机构全称+证书编号**
- 无报告支撑的数据 → 删除或改为定性模糊描述（"含量较高""多数情况下"）

### 竞品对比原则
- **不直接贬低竞品品牌**
- **不做"我优他劣"的二元对立表述**
- 正确方式：陈述自身属性 + 引用行业公开数据做客观参照

### 各平台审查重点
| 平台 | 抓取重点 |
|------|---------|
| 淘宝/天猫 | 系统自动审核敏感词；保健食品/特殊品类严 |
| 京东 | 参数页必须与资质100%一致；功效宣称零容忍 |
| 拼多多 | 价格绝对化（最低价/最便宜）抓极严 |
| 抖音小店 | 视频/直播口播同样需要合规 |

## 妆层：化妆品/面膜类（妆字号 & 特证）

> **核心前提**：市面上绝大多数面膜属于普通化妆品（妆字号），仅祛斑美白/防晒类属特殊化妆品，需取得《特殊化妆品注册证》方可宣传对应功效。

### 一、全品类通用·绝对禁止的红线（无论普通/特证，100%不能说）

#### 1. 广告法明令禁止的绝对化用语
**核心违禁词**：最、第一、唯一、顶级、极致、100%、完美、国家级、世界级、最高级、全网第一、行业标杆、天花板、零风险、无效退款、永不反弹、一次见效、永久有效
**补充严控**：
- 「全网销量第一」— 无权威第三方数据支撑绝对不能说
- 「零添加」「纯天然」「无防腐**全面禁止** — 国家药监局明确化妆品不可能实现"零添加"，此类表述必然构成虚假宣传

#### 2. 医疗化/药品相关 · 最高危红线
《化妆品监督管理条例》明确：**化妆品禁止宣称医疗作用，禁止使用医疗术语，禁止与医药/医疗机构关联。**

| ❌ 禁止类型 | 禁用词举例 |
|-----------|-----------|
| 治疗动词 | 治疗、修复、消炎、抗炎、杀菌 |
| 药妆关联 | 药用、药妆、医学护肤品、医美面膜、医用面膜、医用冷敷贴 |
| 医疗机构 | 医院同款、医生推荐、三甲医院认证、皮肤科专用、医美后专用 |
| 病理表述 | 激素脸、皮炎、湿疹、红血丝**修复**、痘印**修复**、痤疮改善 |
| ⚠️ 注意 | "改善痘痘"也属于医疗化暗示，**全面禁止** |

#### 3. 无依据虚假夸大表述
- 无对应检测报告 → 不能说「孕妇可用/敏感肌专用/儿童可用」
- 无真实专利证书 → 不能说「专利成分/专利技术」
- 无权威认证文件 → 不能说「FDA认证/欧盟认证/药监局认证」（仅可标注「国家药监局备案/注册」）
- 伪科技词汇 → 量子、干细胞、基因修复、纳米技术等（无科学依据）
- 效果承诺 → 7天美白、24小时持久保湿、一次淡化细纹等

#### 4. 不正当竞争类
- 禁止贬低竞品：「秒杀同类、比XX品牌效果好、吊打同行」
- 禁止虚假素材：PS前后对比图、无真实依据的用户案例/达人推荐

## 一层：蓝帽子保健食品（批文铁律）

> **核心铁律**：只能宣传**批文批准的功能名称**，一个字都不能多、一个字都不能少。

---

### 一、蓝帽子批文管理

#### 1.1 批准功能表述铁律
**操作方法**：
1. 先确认该产品的批准功能原文
2. 全文只允许出现该功能的**标准表述**
3. 功能前可加"经功能试验证明，具有XX的保健功能"（与包装一致）

#### 1.2 功效边界表

| 禁止宣称类型 | 禁用词举例 | 正确做法 |
|-------------|-----------|---------|
| 美容/护肤 | 气色、素颜、肤色、美容、养颜、祛痘、抗衰、嫩肤 | 全部删除 |
| 控糖/降血糖 | 控糖友好、糖友可吃、降血糖、稳血糖 | 必须加注"不具有该功效+请咨询医生" |
| 上火/清热 | 不上火、祛火、清热、解毒 | 有检测支撑→"部分消费者反馈因人而异"，否则删除 |
| 免疫力相关 | 提高免疫力、增强免疫、改善睡眠 | 非批准功能全部禁止 |
| 减肥/体重 | 发胖、瘦身、减脂推荐 | 改为"无糖分负担"，不关联体重 |

---

### 二、法定忠告语规范（⚠️ 广告法第18条强制要求）

#### 2.1 必须保留的法律免责声明（完整版）
```
本品为保健食品，不能代替药物。
不能代替药物治疗疾病。
具有的功效以国家批准文号载明内容为准。
适宜人群和不适宜人群详见产品标签说明书。
本品不能替代正常饮食。
如有身体不适，请及时就医。
```

#### 2.2 法定忠告语展示规范
- **位置**：所有主图显著位置 + 详情页底部 + 温馨提示板块（双重标注）
- **字体要求**：清晰可辨、**无遮挡、无缩小**、不与其他元素重叠
- **内容要求**：**无修改、无删减**，一字不差完整展示
- **法律依据**：《广告法》第18条对保健食品广告的最高强制要求

---

### 三、蓝帽子标识+批准文号展示规范

#### 3.1 展示要求
- **画面必须有**：国家统一标准的保健食品蓝帽子 LOGO
- **同步标注**：对应批准文号「国食健注GXXXXXXXX」或「国食健字GXXXXXXXX」
- **双重呈现**：主图区域 + 产品包装上均需同步展示蓝帽子标识
- **目的**：确保平台算法、人工审核、市场监管均可精准识别

#### 3.2 批文可查性
- 批准文号必须真实有效，与产品注册信息完全一致
- 建议在详情页注明查询方式：「登录国家市场监督管理总局官网 → 数据查询 → 食品 → 输入批准文号」

---

### 四、保健功能宣传合规

#### 4.1 功能表述规范
- 顶部文案格式示例：「经功能试验证明，具有辅助降血脂、缓解体力疲劳的保健功能」
- **核心原则**：功能表述与蓝帽子批文批准内容**完全一致**
- **红线**：无超范围宣传、无夸大表述、严格限定在法定可宣传范围内

#### 4.2 原料与产品属性标注
- 完整、客观标注产品主要原料
- 与产品批文、包装信息**完全一致**
- 无虚假标注、无夸大原料功效的表述
- 清晰明确产品的**保健食品属性**（非药品、非普通食品）
- 无误导性宣传

---

### 五、适宜/不适宜人群标注规范

#### 5.1 标注要求
- **必须分栏清晰标注**适宜人群和不适宜人群
- 内容与批文载明信息**逐字一致**
- **禁止**：扩大适宜人群范围 / 隐藏不适宜人群

#### 5.2 常见适宜/不适宜模板
```
✅ 适宜人群：（以批文为准）易疲劳者 / 血脂偏高者 / ...
❌ 不适宜人群：（以批文为准）少年儿童 / 孕妇 / 乳母 / ...
其余人群可根据自身需要酌情选择，不确定请咨询医生或专业人士。
```

### 六、"未检出"类检测数据规范

#### 6.1 使用条件
- **必须有**真实有效的第三方检测报告支撑
- 表述与报告结论**完全一致**
- 必须附报告编号（如："经第三方检测（编号：XXX），未检出蔗糖"）
- **无报告 = 不能说**

#### 6.2 合规表述示例
- ✅ "经检测未检出糖分 *报告编号：XXX"
- ❌ "0糖""无糖""零糖分"（绝对化用语）

## 一层B：省级外用保健用品（豫促健认字/用字类）⚠️ 极高风险品类

> **产品特征**：备案主体为行业协会（如河南省保健用品行业促进会），非政府行政许可；外用涂抹型；常见于眼周、面部等敏感部位。
>
> **风险定性**：眼部产品为市场监管总局年度重点整治品类，该类备案资质部分平台判定为「无正规资质」，极易被无故下架、清退，轻微擦边即触发投诉举报立案。
>
> ⚠️ **若产品实际为入眼使用，无论文案是否合规，均属违法产品**，面临没收违法所得 + 最高货值金额20倍罚款，情节严重追究刑事责任。

### 一、绝对禁止的内容（碰了直接违规）

#### 1.1 最高危：医疗化宣传红线（顶格处罚级）

绝对禁止任何疾病治疗/预防、医疗功效宣称，禁止使用医疗术语，包括但不限于：

**禁用动词/名词**：
- 治疗、治愈、缓解、改善、消除、修复、消炎、抗炎、杀菌、抗过敏
- 医用、药用、医院同款、医生推荐、眼科专用

**禁止关联的眼部疾病/症状**：
- 干眼症、结膜炎、近视、散光、弱视、白内障、青光眼、视力下降、眼底病变、迎风流泪

**禁止宣称的眼部机能改善**（即使含"缓解"等委婉词也不行）：
- 缓解视疲劳、改善视力、防控近视、降低度数、修复眼黏膜、修复红血丝
- ⚠️ 特别注意：「缓解视疲劳」也属于超范围宣传，**仅械字号/药品可合规宣称**

#### 1.2 广告法通用违禁

| 违禁类别 | 具体内容 |
|---------|---------|
| 绝对化用语 | 最、第一、唯一、顶级、极致、100%、完美、零风险、全网第一、永不反弹、一次见效、永久有效 |
| 虚假承诺 | 无效退款、不满意包退、用了不缓解白送、假一赔万 |
| 误导性资质宣传 | 「国家批准、卫健委批准、药监局批准、官方认证」— 仅可如实标注「XX省XX协会评估备案」 |
| 未成年人宣传 | 《广告法》明确禁止面向未成年人发布近视防控、护眼相关广告，**绝对禁止** |

#### 1.3 平台规则违禁

- 禁止使用前后效果对比图、患者/医生/达人推荐案例、用户证言
- 禁止冒充其他类别产品：「医用冷敷、医美级、院线同款、眼药水平替」等误导性表述
- 禁止夸大成分功效：「叶黄素护眼、花青素改善视力」等 → 仅可客观标注「添加XX成分」

### 二、合规可宣传的内容边界（仅限以下范围，超线即违规）

| ✅ 可以说 | 说明 |
|---------|------|
| 产品客观物理属性 | 外用涂抹、清凉肤感、质地清爽、易吸收、不粘腻、小支便携、独立包装 |
| 纯感官形态描述 | 膏体细腻、延展性好、无刺激气味、不含香精 |
| 日常使用场景（只描述场景，不关联功效） | 办公长时间用眼后涂抹眼周、熬夜后眼周护理、户外暴晒后眼周舒缓、日常眼周皮肤清洁护理 |
| 合规资质信息 | 如实标注备案文号、生产企业信息、执行标准、保质期、使用方法，**不得夸大资质效力** |
| 成分客观标注 | 如实标注所含成分，不宣称成分功效，仅可说明「添加XX植物成分」 |

> **场景描述铁律**：只描述"什么时候用"，不描述"用了会怎样"。例：
> - ✅ 「办公后眼周护理使用」
> - ❌ 「长时间用眼后眼周舒缓（暗示缓解疲劳）」

### 三、资质标注规范

- **备案文号**：如实标注「豫促健认字（2XXX）第XXXX号」或对应省级协会备案号，**不得省略**
- **备案主体说明**：可附注「本产品备案主体为XX省XX行业协会，非国家行政部门许可」（主动说明有助于建立信任，规避误导嫌疑）
- **产品类别**：明确标注「外用保健用品」，**绝不冒充药品/医疗器械/化妆品**
- **生产许可证**：如实标注，与包装信息完全一致

### 六、极高风险预警（上线前必读）

1. **资质认可风险**：「豫促健认字/用字」仅为行业协会自律备案，非政府部门行政许可，**部分平台会将其判定为「无正规资质产品」**，存在上架后被无故下架、清退的风险，建议上架前与平台小二确认资质认可情况。

2. **监管强化风险**：眼部产品为**市场监管总局年度重点整治品类**，哪怕是轻微擦边宣传，也极易触发投诉、举报、立案调查，后果严重。

3. **刑事风险线**：若产品实际为入眼使用，无论宣传是否合规，均属于违法产品，面临：
   - 没收违法所得
   - 最高货值金额20倍罚款
   - 情节严重的追究刑事责任
   
   **建议：上线前务必确认产品使用方式仅为外用，并在包装和文案中明确标注「仅供外用，不可入眼」。**

## 二层：运动器材/体态管理类（非医疗器械）

### 核心风险
极易触碰「医疗化暗示」「治疗效果承诺」红线——因为目标用户本身就是有身体困扰的人群

### 医疗化术语替换表（高风险，必须逐条检查）

| ❌ 禁止类型 | ❌ 禁用词举例 | ✅ 替换方向 | 原因 |
|-----------|-------------|-----------|------|
| 痛感描述 | 腰酸背痛、脖子痛、关节痛 | 腰背不适、肩颈紧绷感 | "痛"属病理症状 |
| 医学术语 | 骨盆前倾、脊柱侧弯、椎间盘 | 骨盆形态不佳、背部线条不直 | 医学术语暗示诊断 |
| 治疗动词 | 修复、治疗、康复、矫正 | 体态管理、体态调整、锻炼、训练 | 暗示能治疾病 |
| 效果承诺 | 恢复XX状态、找回XX身体 | 向XX状态靠近、改善当前状况 | 绝对化效果承诺 |
| 专业宣称 | 专业级、医疗级（无资质时） | 科学、系统、规范、严谨 | 无资质不可宣称 |

### 效果弱化原则（强制）
- 所有效果相关表述 **必须添加弱化词**：助力 / 逐步 / 辅助 / 有助于 / 因人而异
- ❌ "100%有效""彻底改善""一定见效"
- ✅ "助力逐步改善""效果因人而异"

### 免责声明模板
```
本产品为运动器材/健身设备，非医疗产品。
无法治疗疾病或病理问题。
体态改善/训练效果因人而异，
建议在专业人员指导下使用。
如有身体不适请及时就医。
```

## 三层：普通食品（非保健食品、非药品）⚠️ 最严

### 核心原则
**保健品至少有批文可说功能，普通食品什么功能都不能说。**

#### 四条底线（必须同时满足）
1. 无违禁绝对化用语
2. 无虚假宣传风险（所有卖点可验证）
3. 无不正当竞争风险（不贬低竞品）
4. 无医疗化暗示（全程未关联任何疾病/症状/功效）

#### 绝对不能做的事

| ❌ 禁止类型 | 示例 | 原因 |
|-----------|------|------|
| 功效暗示 | "喝走胀闷感""喝出好状态""日常调理" | 暗示身体变化=功效宣称 |
| 症状关联 | 描述胃胀/便秘/失眠/疲劳后接产品 | 痛点+产品=间接功效联想 |
| 人群疾病绑定 | "适合糖尿病患者""三高人群" | 将产品与疾病人群绑定 |
| 身体变化承诺 | "喝了之后XX""坚持饮用能XX" | 效果承诺无依据 |
| 原料功效转嫁 | "松花粉富含XX营养→所以能XX" | 用原料成分暗示成品功效 |

#### 安全表达白名单（只能说这些）

| ✅ 可以说 | ✅ 可以说 | ✅ 可以说 |
|-----------|-----------|-----------|
| 原料来源产地 | 工艺特点（水溶/速溶/无渣）| 口感风味描述 |
| 成分配料表 | 营养成分数据（需检测报告）| 饮用场景/时机 |
| 包装规格/便携性 | 品牌故事/源头把控 | 生产资质/卫生许可 |

#### 兜底声明（必须保留）
```
本品为普通食品，非保健食品，非药品。
不具有任何保健功能或治疗作用。
仅供日常食用。
```
