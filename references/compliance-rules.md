# 电商文案合规规则库

> 按产品类型分层，使用时自动匹配对应规则层。**普通食品红线最严——什么功能都不能说。**
>
> 适用法规：《广告法》《化妆品监督管理条例》《化妆品功效宣称评价规范》+淘宝/拼多多/抖音/京东全平台电商规则

---

## 零层：通用规则（所有产品必过）

### 绝对化用语禁用清单
**禁止**：唯一、最、所有、只有、第一、绝对、顶级、极致、完美、无敌、彻底、完全、100%、99%、100倍、国家级、特效
**替换为**：多数 / 约 / 显著 / 较为 / 之一 / 部分 / 大多数 / 助力 / 逐步

### 数据/事实宣称必须有背书
- 每个"XX倍""XX%""未检出""含有" → **必须附第三方检测报告编号**
- 认证类宣称 → **必须附认证机构全称+证书编号**
- 无报告支撑的数据 → 删除或改为定性模糊描述（"含量较高""多数情况下"）

### 竞品对比原则
- **不直接贬低竞品品牌**
- **不做"我优他劣"的二元对立表述**
- 正确方式：陈述自身属性 + 引用行业公开数据做客观参照

### 各平台审查重点
| 平台 | 抓取重点 |
|------|---------|
| 淘宝/天猫 | 系统自动审核敏感词；保健食品/特殊品类严 |
| 京东 | 参数页必须与资质100%一致；功效宣称零容忍 |
| 拼多多 | 价格绝对化（最低价/最便宜）抓极严 |
| 抖音小店 | 视频/直播口播同样需要合规 |

## 妆层：化妆品/面膜类（妆字号 & 特证）

> **核心前提**：市面上绝大多数面膜属于普通化妆品（妆字号），仅祛斑美白/防晒类属特殊化妆品，需取得《特殊化妆品注册证》方可宣传对应功效。

### 一、全品类通用·绝对禁止的红线（无论普通/特证，100%不能说）

#### 1. 广告法明令禁止的绝对化用语
**核心违禁词**：最、第一、唯一、顶级、极致、100%、完美、国家级、世界级、最高级、全网第一、行业标杆、天花板、零风险、无效退款、永不反弹、一次见效、永久有效
**补充严控**：
- 「全网销量第一」— 无权威第三方数据支撑绝对不能说
- 「零添加」「纯天然」「无防腐**全面禁止** — 国家药监局明确化妆品不可能实现"零添加"，此类表述必然构成虚假宣传

#### 2. 医疗化/药品相关 · 最高危红线
《化妆品监督管理条例》明确：**化妆品禁止宣称医疗作用，禁止使用医疗术语，禁止与医药/医疗机构关联。**

| ❌ 禁止类型 | 禁用词举例 |
|-----------|-----------|
| 治疗动词 | 治疗、修复、消炎、抗炎、杀菌 |
| 药妆关联 | 药用、药妆、医学护肤品、医美面膜、医用面膜、医用冷敷贴 |
| 医疗机构 | 医院同款、医生推荐、三甲医院认证、皮肤科专用、医美后专用 |
| 病理表述 | 激素脸、皮炎、湿疹、红血丝**修复**、痘印**修复**、痤疮改善 |
| ⚠️ 注意 | "改善痘痘"也属于医疗化暗示，**全面禁止** |

#### 3. 无依据虚假夸大表述
- 无对应检测报告 → 不能说「孕妇可用/敏感肌专用/儿童可用」
- 无真实专利证书 → 不能说「专利成分/专利技术」
- 无权威认证文件 → 不能说「FDA认证/欧盟认证/药监局认证」（仅可标注「国家药监局备案/注册」）
- 伪科技词汇 → 量子、干细胞、基因修复、纳米技术等（无科学依据）
- 效果承诺 → 7天美白、24小时持久保湿、一次淡化细纹等

#### 4. 不正当竞争类
- 禁止贬低竞品：「秒杀同类、比XX品牌效果好、吊打同行」
- 禁止虚假素材：PS前后对比图、无真实依据的用户案例/达人推荐

### 二、普通妆字号面膜·绝对禁说内容（无特证 = 100%不能说）

| 违禁功效类别 | 绝对禁说词汇 | 宣传前提 |
|-------------|-----------|---------|
| **祛斑美白类** | 美白、淡斑、祛斑、去黄、提亮肤色(关联美白)、淡化黑色素、阻断黑色素、焕白、匀白 | 必须「祛斑美白类特殊化妆品注册证」 |
| **防晒类** | 防晒、防紫外线、隔离紫外线、SPF值、PA值、防晒伤、防晒黑、防光老化 | 必须「防晒类特殊化妆品注册证」 |
| **功效宣称类** | 抗皱、紧致、舒缓、控油、去角质、**修护**、保湿 | 必须有《化妆品功效宣称评价报告》，无报告绝对不能宣传 |

> ⚠️ **注意**：即使"修护""保湿"这类看起来很基础的词，没有功效评价报告也不能作为功效宣称使用。

### 三、90%商家踩坑·擦边球违禁表述

#### 高频踩坑替换表

| ❌ 违禁表述 | ✅ 合规替代话术 | 触发风险 |
|------------|---------------|---------|
| **修复**红血丝/激素脸/痘印 | 修护肌肤屏障、维稳肌肤状态 | 🔴 "修复"是医疗术语 |
| **消炎**、抗炎、消肿、退红 | 舒缓泛红不适、维稳肌肤状态 | 🔴 医疗术语 |
| **抗衰**老、抗初老、冻龄、逆龄 | 改善干纹细纹、紧致肌肤轮廓、提升肌肤弹性 | 🔴 虚假夸大/医疗化 |
| **抗过敏**、敏感肌救星 | 温和低刺激、适配敏感肌日常护理 | 🟡 需刺激性检测报告 |
| **去黄**、提亮、**美白** | 改善肌肤暗沉、匀净肤色、焕亮肌肤光泽 | 🔴 特证才可以说 |
| **祛痘**、淡化痘印 | 调节水油、改善毛孔堵塞、清洁老废角质 | 🔴 医疗化暗示 |
| 100%纯度、高浓度 | 标注合规成分名称及添加量 | 🟡 夸大成分效果 |
| **医美面膜**、医用面膜、冷敷贴 | 妆字号日常护理面膜、温和低刺激护理款 | 🔴 冒用医疗器械类别 |
| 一次见效、7天焕白 | 坚持使用可改善肌肤状态、日常护理维稳 | 🔴 效果承诺 |

#### 各平台额外拦截重点

| 平台 | 自动拦截关键词 |
|------|--------------|
| **淘宝/天猫** | 医美面膜、医用冷敷贴、械字号、天花板、YYDS、首选、顶级、独家、无效退款 |
| **拼多多** | 价格绝对化 + 所有极限词 + 承诺性表述 |
| **抖音小店** | 一夜回春、换脸、人皮面膜、童颜等暗示医疗整形效果的话术；直播口播同样审查 |
| **京东** | 参数页必须与资质100%一致；功效宣称零容忍；冒充械字号重罚 |

### 四、合规表达白名单（可直接使用）

| ✅ 可以说 | ✅ 可以说 | ✅ 可以说 |
|-----------|-----------|-----------|
| 成分名称+添加量（有配方表支撑） | 产品质地/肤感描述 | 使用方法/步骤 |
| 生产厂家信息 | 备案/注册编号 | 包装规格 |
| 适用肤质（需有测试支撑） | 品牌故事 | 用户真实反馈（非PS） |
| 工艺特点 | 研发团队背景 | 专利号（需有证书） |

> **原则**：只说**产品有什么**（成分、工艺、资质、专利号），不说**用了之后会怎样**（变白、变嫩、皱纹消失）。

### 五、化妆品必保留的兜底声明
```
本品为化妆品，非药品，非医疗器械。
不具有治疗疾病或改善病理问题的作用。
具体功效以国家药监局备案/注册信息为准。
效果因人而异，建议先做局部测试。
```

---

### 核心铁律
只能宣传**批文批准的功能名称**，一个字都不能多、一个字都不能少。

**操作方法**：
1. 先确认该产品的批准功能原文
2. 全文只允许出现该功能的**标准表述**

### 功效边界表

| 禁止宣称类型 | 禁用词举例 | 正确做法 |
|-------------|-----------|---------|
| 美容/护肤 | 气色、素颜、肤色、美容、养颜、祛痘、抗衰、嫩肤 | 全部删除 |
| 控糖/降血糖 | 控糖友好、糖友可吃、降血糖、稳血糖 | 必须加注"不具有该功效+请咨询医生" |
| 上火/清热 | 不上火、祛火、清热、解毒 | 有检测支撑→"部分消费者反馈因人而异"，否则删除 |
| 免疫力相关 | 提高免疫力、增强免疫、改善睡眠 | 非批准功能全部禁止 |
| 减肥/体重 | 发胖、瘦身、减脂推荐 | 改为"无糖分负担"，不关联体重 |

### 必须保留的法律免责声明
```
本品为保健食品，不能代替药物。
不能代替药物治疗疾病。
具体功效以批准文号载明内容为准。
```

## 二层：运动器材/体态管理类（非医疗器械）

### 核心风险
极易触碰「医疗化暗示」「治疗效果承诺」红线——因为目标用户本身就是有身体困扰的人群

### 医疗化术语替换表（高风险，必须逐条检查）

| ❌ 禁止类型 | ❌ 禁用词举例 | ✅ 替换方向 | 原因 |
|-----------|-------------|-----------|------|
| 痛感描述 | 腰酸背痛、脖子痛、关节痛 | 腰背不适、肩颈紧绷感 | "痛"属病理症状 |
| 医学术语 | 骨盆前倾、脊柱侧弯、椎间盘 | 骨盆形态不佳、背部线条不直 | 医学术语暗示诊断 |
| 治疗动词 | 修复、治疗、康复、矫正 | 体态管理、体态调整、锻炼、训练 | 暗示能治疾病 |
| 效果承诺 | 恢复XX状态、找回XX身体 | 向XX状态靠近、改善当前状况 | 绝对化效果承诺 |
| 专业宣称 | 专业级、医疗级（无资质时） | 科学、系统、规范、严谨 | 无资质不可宣称 |

### 效果弱化原则（强制）
- 所有效果相关表述 **必须添加弱化词**：助力 / 逐步 / 辅助 / 有助于 / 因人而异
- ❌ "100%有效""彻底改善""一定见效"
- ✅ "助力逐步改善""效果因人而异"

### 免责声明模板
```
本产品为运动器材/健身设备，非医疗产品。
无法治疗疾病或病理问题。
体态改善/训练效果因人而异，
建议在专业人员指导下使用。
如有身体不适请及时就医。
```

## 三层：普通食品（非保健食品、非药品）⚠️ 最严

### 核心原则
**保健品至少有批文可说功能，普通食品什么功能都不能说。**

#### 四条底线（必须同时满足）
1. 无违禁绝对化用语
2. 无虚假宣传风险（所有卖点可验证）
3. 无不正当竞争风险（不贬低竞品）
4. 无医疗化暗示（全程未关联任何疾病/症状/功效）

#### 绝对不能做的事

| ❌ 禁止类型 | 示例 | 原因 |
|-----------|------|------|
| 功效暗示 | "喝走胀闷感""喝出好状态""日常调理" | 暗示身体变化=功效宣称 |
| 症状关联 | 描述胃胀/便秘/失眠/疲劳后接产品 | 痛点+产品=间接功效联想 |
| 人群疾病绑定 | "适合糖尿病患者""三高人群" | 将产品与疾病人群绑定 |
| 身体变化承诺 | "喝了之后XX""坚持饮用能XX" | 效果承诺无依据 |
| 原料功效转嫁 | "松花粉富含XX营养→所以能XX" | 用原料成分暗示成品功效 |

#### 安全表达白名单（只能说这些）

| ✅ 可以说 | ✅ 可以说 | ✅ 可以说 |
|-----------|-----------|-----------|
| 原料来源产地 | 工艺特点（水溶/速溶/无渣）| 口感风味描述 |
| 成分配料表 | 营养成分数据（需检测报告）| 饮用场景/时机 |
| 包装规格/便携性 | 品牌故事/源头把控 | 生产资质/卫生许可 |

#### 兜底声明（必须保留）
```
本品为普通食品，非保健食品，非药品。
不具有任何保健功能或治疗作用。
仅供日常食用。
```

## 上线前检查清单（全品类通用）

1. 全文搜索绝对化禁用词（零容忍）
2. 确认功效/功能宣称与资质批准范围一致（不多不少）
3. 确认每条数据/认证宣称附有报告编号或来源
4. 确认所有效果/改善类表述均有弱化词
5. 确认页面底部有法律免责声明且字号可辨识
6. 确认不适宜人群/注意事项与资质文件一致
7. 备案材料齐全（资质批件、检测报告、生产许可等）
8. 提交平台预审，通过后再正式上线
